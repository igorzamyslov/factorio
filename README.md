# Factorio scripts

Contains useful scripts for factorio


## Requirements

- Python 3.7


## Running the scripts

```bash
bash <path_to_script>
```


## Obtaining your own game data

Game data is obtained using the mod "Data Exporter to JSON"

1. Install the mod
2. Start new game
3. Game data is created under `%APPDATA%\Factorio\script-output\recipe-lister`
